简答题
1.简述对面向对象编程的理解
面向对象是一种很自然的思想，符合我们的认知特点。例如在游戏开发中，要创造一个鱼塘，鱼塘中有不同品种的鱼和水底的石头，而鱼和石头是完全不同的两类物体，鱼虽然也有很多类别但都有共同的特征，根据面向对象编程的思想，我们可以抽象出鱼和石头两个不同的类，通过继承父类创造出不同品种的鱼，再通过实例化创造具体的鱼对象和具体的石头对象。面向对象编程就是如此，以自然的认知方式，抽象出不同的对象，将对象视为组成单元，把数据和函数包裹在其中，通过对象之间的数据传递来执行程序。

2.比较定义、声明和初始化三个概念
定义：为变量分配存储空间
声明：表明变量的类型和名字，一般不用为变量分配存储空间
初始化：最开始的赋值，一般在定义时同时赋值进行初始化

3. 试比较 char*, char[], string 之间的区别
char*是指向字符串的指针，char[]是字符串数组，很多情况下可以混用，但前者是一个指针变量，后者是地址常量
而string是C++提供的字符串类

4. 试比较取成员类型的运算符 . 与->的区别
点号(.)左边必须为实体，箭头(->)左边必须为指针
例如：
struct Node
{
	int member;
}
两种访问方式：
(1)Node s;s.member=1;
(2)Node *s;s->member=1;//(或(*s).member=1)
